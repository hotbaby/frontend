jQuery AJAX

jQuery AJAX load

load() 方法从服务器加载数据，并把返回的数据放入被选的元素中

语法

$(selector).load(url, data, callback)
  • url 必选参数
  • data 可选参数
  • callback 可选参数

callback 参数

  • responseTxt 包含返回成功的结果内容
  • statusTxt 包含调用的状态
  • xhr 包含XMLHttpRequest对象

jQuery $.get方法

$.get() 方法通过HTT GET请求从服务器请求数据

语法

$.get(URL, callback)

实例

$("button").click(function(){
    $.get("http://mengyangyang.org", function(data, status){
        alter("Data: " + data + "\n Status: " + status)
    })
})

jQuery $.post方法

$.post() 方法通过HTTP POST 向服务器提交数据

语法

$.post(URL, data, callback)

实例

$("button").click(function(){
    $.post("http://mengyangyang.org", 
    {
        name: "Donald Duck",
        city: "Duckburg"
    },
    function(data, status){
        alter("Data: " + data + "\n Status: " + status)
    })
})
