jQuery 介绍

jQuery库 特性

jQuery 是一个JavaScript库。

  • HTML 元素选取
  • HTML 元素操作
  • CSS 操作
  • HTML 事件函数
  • JavaScript 特效和动画
  • HTML DOM遍历和修改
  • AJAX
  • Utilities

向页面中添加jQuery库

<script type="text/javascript" src="https://code.jquery.com/jquery-3.1.1.slim.min.js"><script>
