id 选择器

id 选择器可以为标有特定 id 的 HTML 元素指定特定的样式。 id 选择器以 "#" 来定义。 
#red {color: red;}
#green {color: green;}
注意: id 属性只能在每个 HTML 文档中出现一次。

id 选择器和派生选择器

在现代布局中，id 选择器常常用于建立派生选择器。 
#sidebar p {
	font-style: italic;
	text-aling: right;
	margin-top: 0.5em;
}

上面的样式只会应用于出现在 id 是 sidebar 的元素内的段落。这个元素很可能是 div 或者是表格单元，尽管它也可能是一个表格或者其他块级元素
